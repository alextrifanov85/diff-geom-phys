# CONTRIBUTING

Приветствую всех в репозитории!

Здесь публикуются исходники конспектов семинаров в формате LaTeX.  
Если вы заметили опечатку, неточность, неудачную формулировку, проблему 
в оформлении или хотите предложить улучшение, пожалуйста, отправьте изменения через **Pull Request**.


## Как вносить правку 

1. Сделайте **fork** репозитория.
2. Клонируйте свой fork:
```bash
    git clone https://github.com/alextrifanov85/diff-geom-phys.git
    cd diff-geom-phys
```
3. Создайте отдельную ветку:
```bash
    git checkout -b fix-typos-seminar-01
```
4. Внесите изменения.
5. Сделайте коммит:
```bash
    git add .
    git commit -m "Исправлены опечатки в seminar01.tex"
```
6. Отправьте ветку в свой fork:
```bash
    git push origin fix-typos-seminar-01
```
7. Откройте **Pull Request** в ветку main основного репозитория.

## Что приветствуется

- исправление опечаток;
- исправление неточностей;
- уточнение формулировок;
- улучшение структуры и верстки;
- унификация обозначений;

## Общие рекомендации:

- не работайте напрямую в *main*
- не объединяйте много несвязных правок в один PR;
- кратко описывайте, что именно изменено.
